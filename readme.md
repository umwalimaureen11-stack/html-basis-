# HTML Fundamentals Assignment

Project Description
A multi-page website built using HTML (and optional CSS) to demonstrate core web development fundamentals including tables, text formatting, lists, images, forms, and multi-page navigation.

## HTML Pages Included
- `index.html` — Home page with ordered/unordered lists, images, and a student registration form
- `about.html` — About page describing the project and student info
- `table.html` — Data table of student records and an HTML text formatting reference table

## What This Project Demonstrates
- Correct HTML document structure and semantic elements
- Tables with `<caption>`, `<thead>`, and `<tbody>`
- Inline text formatting tags (`<b>`, `<i>`, `<u>`, `<strong>`, `<em>`, `<mark>`)
- Ordered and unordered lists
- Image embedding with `alt`, `width`, and `height` attributes
- An image used as a hyperlink
- A registration form with text input, email input, radio buttons, checkboxes, and a submit button
- Multi-page navigation using `<a>` tags with `title` tooltips
- External CSS styling via `style.css`
